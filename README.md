# Dyslexia Reading Assistant

## Project Description

Dyslexia Reading Assistant is a web application designed to help individuals with dyslexia better understand and read complex text. The application simplifies difficult sentences into clear, easy-to-understand explanations and provides audio playback of the simplified content to support reading comprehension.

In addition to text simplification, the application assists users in learning the pronunciation of difficult words, helping improve reading confidence and language skills.

## Features

* Simplifies complex sentences into easier language.
* Generates clear explanations for difficult text.
* Converts simplified explanations into audio for listening support.
* Provides pronunciation assistance for challenging words.
* User-friendly web interface built with Streamlit.

## Technology Stack

* **Python** – Backend development
* **Streamlit** – Web application framework
* **Groq API** – AI inference platform
* **Llama Model** – Text simplification and pronunciation assistance

## How It Works

1. The user pastes a complex sentence or paragraph into the application.
2. The Groq-hosted Llama model analyzes the text and generates a simplified explanation.
3. The simplified explanation is displayed to the user.
4. An audio version of the simplified explanation is generated for listening support.
5. Users can also obtain pronunciation guidance for difficult words.

## Use Cases

* Supporting individuals with dyslexia in reading and comprehension.
* Assisting students who struggle with complex language.
* Improving pronunciation and vocabulary learning.
* Providing an accessible reading aid for educational purposes.

## Deployment

The application is deployed using Streamlit and built entirely in Python.
